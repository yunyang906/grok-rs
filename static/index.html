会话粘滞

同一个 Claude Code 会话优先固定到同一账号，不可用时自动切换。

ⓘ账号优先级始终先于调度模式：只使用当前最高优先级的可用账号；多个账号优先级相同时，再按均衡轮询或优先用满进行选择。额度耗尽或账号冷却时会自动故障切换。

安全策略

启用流量保护

包含流式请求；建议从 4–8 开始，根据账号数量调整。

平台侧保护

风险控制只能减少异常调用风险，不能保证账号不会被平台限制。
